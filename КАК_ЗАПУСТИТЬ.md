# Как запустить проект «Восточный Лист»

## Требования

- **Python 3.10+** (проверка: `python --version`)
- Терминал (PowerShell или CMD в Windows)

---

## Пошаговая инструкция

### 1. Откройте терминал и перейдите в папку Django-проекта

```bash
cd путь\к\проекту\Web_And_Rasrab-main\myproject
```

Пример для вашего компьютеру:
```bash
cd c:\Users\Bublik\Desktop\Web_And_Rasrab-main\myproject
```

---

### 2. Создайте виртуальное окружение (только при первом запуске)

```bash
python -m venv venv
```

---

### 3. Активируйте виртуальное окружение

**Windows (PowerShell или CMD):**
```bash
venv\Scripts\activate
```

**macOS / Linux:**
```bash
source venv/bin/activate
```

В начале строки должно появиться `(venv)` — значит окружение активно.

---

### 4. Установите зависимости (только при первом запуске)

```bash
pip install -r requirements.txt
```

---

### 5. Примените миграции базы данных (только при первом запуске)

```bash
python manage.py migrate
```

---

### 6. Заполните каталог тестовыми товарами (только при первом запуске)

```bash
python manage.py seed
```

Будут созданы категории чая и 6 тестовых товаров.

---

### 7. Запустите сервер

```bash
python manage.py runserver
```

В консоли появится что-то вроде:
```
Starting development server at http://127.0.0.1:8000/
Quit the server with CTRL+BREAK.
```

---

### 8. Откройте сайт в браузере

Перейдите по адресу: **http://127.0.0.1:8000/**

Остановить сервер: **Ctrl+C** в терминале.

---

## Админ-панель (по желанию)

Чтобы управлять товарами и категориями через веб-интерфейс:

1. Создайте учётную запись администратора (один раз):
   ```bash
   python manage.py createsuperuser
   ```
   Введите логин, email и пароль по запросу.

2. Запустите сервер (`python manage.py runserver`) и откройте:
   **http://127.0.0.1:8000/admin/**

3. Войдите под созданным логином и паролем.

## Возможные проблемы

| Проблема | Решение |
|----------|---------|
| `python` не найден | Установите Python с python.org или используйте `py` вместо `python` |
| `No module named 'django'` | Активируйте venv (`venv\Scripts\activate`) и выполните `pip install -r requirements.txt` |
| Ошибки про миграции | Выполните `python manage.py makemigrations`, затем `python manage.py migrate` |
| Порт 8000 занят | Запустите на другом порту: `python manage.py runserver 8080` и откройте http://127.0.0.1:8080/ |
